Marcado

Aquí es donde describiria cualquier cuestión relativa al marcadoy como se utiliza. Es decir, qué podemos hacer con mark como programadoras, utilizándolo en para la maquetación de una página web. En este primer párrafo he seleccionado unos márgenes de 40px en ambos lados, para que el texto quede encuadrado dentro de la página de una forma más estética.

Este segundo párrafo lo hago únicamente para completar el ejercicio, también queda con los mismos parametros establecidos como márgenes en el anterios, ya que se encuentra dentro de una etiqueta de contenido, la cual al cambiar el estilo, se aplica a todos sus elementos.
